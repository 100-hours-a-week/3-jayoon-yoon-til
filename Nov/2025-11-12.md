## 오늘의 도전 과제와 해결 방법

- 도전 과제 1: S3 권한 문제 해결
```
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "Statement1",
            "Effect": "Allow",
            "Principal": "*",
            "Action": [
                "s3:GetObject",
                "s3:PutObject"
            ],
            "Resource": "arn:aws:s3:::jayoon-bucket/*"
        }
    ]
}
```
